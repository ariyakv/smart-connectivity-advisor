# Smart Connectivity Advisor

Smart Connectivity Advisor is an algorithm-powered web application that helps households find affordable and reliable internet and mobile plans based on their needs.

This project was built as a hackathon project focused on social good, aiming to reduce digital inequality by helping users make informed connectivity decisions.

---

## 🚀 Features

- Personalized plan recommendations based on:
  - Budget
  - Household size
  - Device count
  - Internet usage (school, streaming, gaming, etc.)
- Estimated speed requirements
- Affordability scoring (0–100)
- Reliability risk scoring
- Community need scoring
- Intelligent explanation of recommendations

---

## 🧠 How It Works

The system uses a rule-based recommendation engine to:
1. Estimate required internet speed
2. Match the user to the most suitable plan
3. Evaluate affordability and reliability
4. Generate a clear explanation of the recommendation

This approach ensures:
- fast performance
- explainable results
- realistic decision-making

## ▶️ How to Run the Project

### 1. Clone the repository
git clone https://github.com/yourusername/smart-connectivity-advisor.git  
cd smart-connectivity-advisor  

### 2. Build the project
mvn clean install  

### 3. Run the application
mvn spring-boot:run
